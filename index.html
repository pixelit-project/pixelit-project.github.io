# Supported hardware

You can use the PixelIt with different hardware. You can use a prebuild hardware or build your own hardware. The PixelIt is designed to be expandable. You can add sensors, buttons, displays, etc. to your PixelIt.

# DIY Hardware

  • MCU
    • Wemos D1 mini
    • NodeMCU v2
    • Generic ESP8266
  • Matrix
    • Pixelmatrix (8x32)
    • Pixelmatrix (CJMCU)(4x 8x8)
    • MicroMatrix by foorschtbar (4x 8x8)
  • Extensions
    • LDR GL5516
    • DHT22
    • BMP280
    • BME680
    • DFPlayer Mini
    • Touch Buttons

# Getting started

  • Start with the hardware and checkout the parts list.
  • Then you can flash the firmware and setup the matrix in the webinterface.
  • At last step, you send your own screens to the API or use Node-RED (recommanded) or ioBroker to create and send your own screens to your PixelIt.
