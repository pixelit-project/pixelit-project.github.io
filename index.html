# Supported hardware

  • MCU
    • Wemos D1 mini
    • NodeMCU v2
    • Generic ESP8266
  • Matrix
    • Pixelmatrix (8x32)
    • Pixelmatrix (CJMCU)(4x 8x8)
    • MicroMatrix by foorschtbar (4x 8x8)
  • Extensions
    • LDR GL5516
    • DHT22
    • BMP280
    • BME680
    • DFPlayer Mini
    • Touch Buttons

# Getting started

  • Start with the hardware and checkout the parts list.
  • Then you can flash the firmware and setup the matrix in the webinterface.
  • At last step, you send your own screens to the API or use Node-RED (recommanded) or ioBroker to create and send your own screens to your PixelIt.

# Support

Any problems? Checkout the Troubleshooting section. If you still need help or just connect to other PixelIt users, checkout your forum or the Telegram and Discord groups.
